# dotfiles

Collection of my dotfiles.

Based on [olimorris' dotfiles](https://github.com/olimorris/dotfiles). Thanks
for the inspiration.

## Instructions

- Full install:

  ```shell
  $ rake install
  ```

- Backup:

  ```shell
  $ rake backup
  ```

- Update and backup:

  ```shell
  $ rake sync
  ```
